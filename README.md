
# Chatting App


 A simple chatting app using Nodejs, mongoDB.

### Prerequisits
Chatting app requires [Node.js](https://nodejs.org/) to run.

### Installation
- Download the zip file or clone the repo.
- Start the server.




```sh
$ npm run start
```
-Go to the url in the browser and type
```sh
http://localhost:3000
```


### Technologies
-Nodejs
-Expressjs
-MongoDB
-HTML5
-CSS3
